Edgecutter - Firmware Upgrade Guide

Upgrade procedure

Things you will need

  • an Edgecutter in a suitably powered Eurorack
  • a patch cable
  • an audio player with loud headphone out (a mobile phone is excellent for this)
  • The firmware file:
    (warning, do not play this wav file over your speakers - this file is loud!)
    Download as zipped WAV

Upgrading

  • as always: READ ALL THE INSTRUCTIONS FIRST!

Flashing the new firmware

  1. Power down your Eurorack.
  2. Disconnect all patch cables from the Edgecutter.
  3. Put the "A" knob in center position.
  4. Put the firmware .wav file on an audio player.
  5. A phone or laptop can be used. Make sure the audio player is NOT connected to WIFI/mobile services. If you can, please use airplane mode. If any sound interrupts the playback of the file, the firmware update will fail/be faulty
  6. Disable all postprocessing on the audio playback. This means no EQ, bass boost, etc. Some audio playback programs (like winamp) have resamplers that can make transfer problematic sometimes. Windows media player seems fine.
  7. Connect the audio player's headphone socket to the "A" port on the Edgecutter with a patchcable.
  8. While holding the MODE (the one that selects between gate/trigger and loop mode) button down, power up your Eurorack.
  9. The Edgecutter should not show its normal boot-cycle, but instead it is showing incoming audio level.
  10. After about a second, let go of the MODE button.
  11. Turn up the volume on your audio player to maximum.
  12. Play the Firmware audio file in to the Edgecutter. Make sure the entire file plays! 3 minutes and 18 seconds.
  13. A few seconds after you started the audio file the LEDs in the LED bar will start lighting up one by one to show the progress.
  14. When the entire audio file has played, the Edgecutter will reboot itself. You will see the normal startup LEDs and you can start playing with your new firmware!
  15. If anything goes wrong during playback of the audio file, one or more of the LEDs in the bar will start and continue to blink:
    Try to adjust the volume of the player and check if the "A" knob is centered, power down and start again. Also make sure that the ENTIRE file has been played. If this still does not work - please try another audio player, as some devices have post-processing on the audio which might impede correct transfer. If problems persist, please contact us.
